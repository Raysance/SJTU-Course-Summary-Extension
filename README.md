# SJTU 课堂概要导出

用于导出 SJTU Canvas 课堂视频页的课程概要，并可用 DeepSeek 生成按日期整理的考试要点。

## 安装

1. 下载并解压本项目。
2. 打开 Chrome，进入 `chrome://extensions`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目中的 `chrome-extension` 文件夹。

更新版本时，在 `chrome://extensions` 里点击该扩展的“重新加载”，然后刷新课程视频页。

## 使用方法

1. 在 Chrome 中登录课程平台，并打开课堂视频页面。
2. 点击 Chrome 工具栏中的“SJTU 课堂概要导出”扩展图标。
3. 在扩展页面点击“重新读取”，确认课程和回放数量显示正常。
4. 如需 AI 总结，保持“使用 DeepSeek”开启，填入 API Key 后点击“长期保存”。
5. 在“导出范围”中选择：
   - `全选`：导出所有日期的课程概要；
   - 某一天：只导出当天的课程概要。
6. 点击“导出所选范围”。
7. 等待处理完成后，到 Chrome 下载目录查看 Markdown 文件。

## 导出文件

扩展会在下载目录中生成课程文件夹，通常包含：

- `课程概要索引.md`
- `YYYY-MM-DD-考试要点与课堂概要.md`

如果关闭“使用 DeepSeek”，文件名会变为：

- `YYYY-MM-DD-课堂概要.md`

## 常见问题

- 找不到课程页面：确认课堂视频页已经在 Chrome 中打开，并且已经登录。
- 读取不到回放：刷新课堂视频页，等页面加载完成后再点“重新读取”。
- 某一天导出失败：在“导出范围”中单独选择这一天，再重新导出。
- DeepSeek 总结失败：检查 API Key、余额和网络；也可以关闭“使用 DeepSeek”后只导出平台概要。
