# SJTU 课程资料助手

用于从已登录的 SJTU Canvas 自动识别课堂视频并导出课程概要，可用 DeepSeek、ChatGPT、Claude 或 Gemini 生成按日期整理的课堂笔记；也支持读取课程“文件”页面，勾选课件后一键打包下载 ZIP。

## 安装

1. 下载并解压本项目。
2. 打开 Chrome，进入 `chrome://extensions`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择项目中的 `chrome-extension` 文件夹。

更新版本时，在 `chrome://extensions` 里点击该扩展的“重新加载”，然后刷新课程视频页。

## 使用方法

1. 在 Chrome 中登录 `https://oc.sjtu.edu.cn/`。
2. 点击 Chrome 工具栏中的“SJTU 课程资料助手”扩展图标。
3. 在扩展页面点击“重新读取”，扩展会通过 Canvas API 获取本学期课程列表；识别到多个课程时，可在顶部选择目标课程。也可以先打开课程的 `文件` 页面，再点击扩展图标，扩展会优先定位到该课程。

## 课程文件打包下载

选择课程后，“课程文件”模块会自动读取该课程文件。可搜索文件名或文件夹路径，勾选需要的文件后点击打包下载按钮，扩展会生成一个 ZIP 文件并保留课程文件夹路径。ZIP 文件名会使用当前选择的课程名。遇到 Canvas API 不可用时会回退到当前课程文件页的实际 DOM 识别。

## 课堂视频概要生成

1. 如需 AI 总结，点击右上角“AI 设置”，在弹窗中选择供应商和模型，填入 API Key 后点击“保存”。
2. 可在“AI 设置”中点击“测速”；导出时会默认先自动测速。
3. 可选择默认模板：默认、期末复习或 LaTeX 笔记；也可在“追加 Prompt”中写入自己的整理要求。
4. 在“课堂视频”模块中勾选要导出的日期，也可以点击“全选”。
5. 鼠标悬浮在某一天上，可预览当天字幕。
6. 点击“导出所选范围”。
7. 处理完成后，点击日志栏中的下载按钮保存 ZIP。

## 导出文件

扩展会在下载目录中生成一个 ZIP 文件，解压后通常包含：

- `导出索引.md`
- `AI整理/YYYY-MM-DD-课堂笔记.md`
- `AI整理/YYYY-MM-DD-复习资料.md`
- `AI整理/YYYY-MM-DD-LaTeX 笔记.tex`
- `原字幕/YYYY-MM-DD-原字幕.md`

如果关闭“使用 AI”，ZIP 中不会生成 `AI整理/` 文件，只会导出原字幕。

## 常见问题

- 找不到课程页面：确认 Chrome 已经登录 `https://oc.sjtu.edu.cn/`。
- 读取不到回放：重新登录课程平台后再点“重新读取”。
- 某一天导出失败：在“导出范围”中只勾选这一天，再重新导出。
- AI 总结失败：检查供应商、模型名、API Key、余额和网络；也可以关闭“使用 AI”后只导出原字幕。
